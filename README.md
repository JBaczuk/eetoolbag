# Circuit Toolbag

## Installation
```
npm install eetoolbag
```

## Usage
### Find Voltage Divider
Finds the best voltage divider using standard resistances and the specified tolerance and max output impedance. 
```
ee gv
```

## Test
`npm run test`

## Publishing
```
npm publish
```

## Development
```
npm run build-cli
```
